# Tests

This directory contains tests for the `draw-mermaid-diagrams` skill.

## Test Cases

1. **Syntax Validation** — Verify generated Mermaid code has balanced brackets, valid arrows, and proper keywords.
2. **Diagram Type Selection** — Verify the correct diagram type is chosen for the given intent.
3. **Readability Rules** — Verify edge labels are concise, direction is consistent, and IDs are unique.

Add test implementations here as the skill evolves.

# References

This directory contains reference documentation for the shadcn/ui component integration skill.

## Files

- `setup-guide.md` — Step-by-step project initialization and configuration verification.
- `component-catalog.md` — Complete component reference with props and dependencies.
- `customization-guide.md` — Theming, variant patterns, and extension best practices.
- `migration-guide.md` — Upgrading from other UI libraries or older shadcn/ui versions.

## When to Read

Read these references when setting up a new project, troubleshooting installation errors, or customizing component themes and variants.

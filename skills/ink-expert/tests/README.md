# Tests

This directory contains test cases and validation scripts for the Ink Expert skill.

## Test Cases

1. **Version Matrix Detection** — Confirm correct detection of Node, Ink, and React versions.
2. **Migration Accuracy** — Given an Ink v5 codebase, verify the v6 migration produces valid code.
3. **Component Ecosystem** — Validate that recommended third-party components are compatible with the detected Ink version.
4. **Testing Patterns** — Ensure ink-testing-library is configured and frame assertions pass.

Use `ink-testing-library` for behavior-level tests in real projects.

# References

This directory contains reference documentation for the screenshot capture skill.

## Files

- `capture-playbook.md` — Decision rules for choosing capture modes (full screen, app, region, active window) across platforms.
- `environment-matrix.md` — Platform-specific behavior, tool availability, and permission handling.
- `troubleshooting-guide.md` — Common capture failures and resolutions per OS.

## When to Read

Read these references when you encounter permission errors, multi-display issues, or need to select the right capture tool for a specific platform.

# Prompt Review Rules

## Must-Have Checks

- Objective is specific and testable.
- Input context is explicit.
- Editable boundaries are listed.
- Deliverables are concrete.
- Validation commands are included.
- Risky assumptions are surfaced.

## Common Weaknesses

- Generic instructions with no repository context.
- Missing acceptance criteria.
- No mention of existing conventions or guardrails.
- No verification plan.

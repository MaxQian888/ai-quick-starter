# Tests

This directory contains tests for the `enhance-prompt` skill.

## Test Cases

1. **Vague Input Enhancement** — Verify that a vague input like "make me a login page" becomes a structured prompt with all required sections.
2. **DESIGN.md Integration** — Verify that when DESIGN.md exists, its design system block is correctly injected.
3. **Targeted Edit Enhancement** — Verify that feature requests like "add a search bar" produce specific, actionable change descriptions.

Add test implementations here as the skill evolves.

# Tests

This directory contains tests for the `design-md` skill.

## Test Cases

1. **Token Extraction** — Verify that color tokens, typography, and spacing are correctly extracted from sample HTML.
2. **Schema Compliance** — Verify that generated `DESIGN.md` files match the required section schema.
3. **Unknown Fact Handling** — Verify that uncertain observations are marked as unknown rather than fabricated.

Add test implementations here as the skill evolves.

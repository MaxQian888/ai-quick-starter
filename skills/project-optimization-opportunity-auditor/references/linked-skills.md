# Linked Skills

## Routing Rules

### `$project-architecture-design-analyzer`
Use when backlog items involve module boundaries, dependency seams, or structural layering.

### `$feature-gap-requirements-auditor`
Use when docs/specs/plans exist and backlog includes docs-vs-code alignment risk.

### `$build-project-fixer`
Use when backlog includes missing or weak lint/test/build command surfaces.

## Anti-Overlap

- This skill prioritizes opportunities.
- Linked skills perform deeper diagnosis or implementation guidance.
- Do not duplicate deep analysis that linked skills already specialize in.

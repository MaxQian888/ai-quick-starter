# Grader

## Purpose

Evaluate whether the agent correctly retrieved Stitch screens, built a Remotion composition with proper transitions and overlays, and rendered a walkthrough video following best practices.

## Scoring Criteria

1. **Screen Retrieval** (25%): Were Stitch project and screen IDs discovered, and screenshots downloaded?
2. **Composition Setup** (25%): Was a Remotion project initialized or reused, with proper component structure?
3. **Transitions & Overlays** (20%): Were professional transitions, text overlays, and timing applied?
4. **Preview & Refine** (15%): Was Remotion Studio used for preview before final render?
5. **Asset Organization** (15%): Were screens organized in a manifest and assets kept in predictable paths?

## Pass Threshold

Score >= 75% to pass.

# wechat-zh：节奏画面型（公众号 / 自媒体 / 中长文叙事）

目标读者：通勤地铁、午饭间隙刷手机的真人。
参考调性：双雪涛、于轩《刷屏》、晚点 LatePost、虎嗅深度稿。

## 核心立场

- **节奏 > 逻辑完美**
- **画面 > 概念**
- **作者就是这个人**（不藏第一人称）
- 段段要有"接着读下一段"的钩子

## 句法偏好

- **短句多**，一段 1-3 句的"散段"频繁出现
- 单字句、句子片段可以用：**"散了。"**、**"懂的都懂。"**、**"就这。"**
- 长句也行，但要有节奏（用逗号断句切碎，不要一气呵成）
- 一段一画面，画面之间靠节奏连，不靠"此外"、"另外"

## 必须出现

- 至少 1 个**第一人称**视角（"我朋友"、"我那天"、"我前几年"）
- 至少 1 个**画面感细节**（具体场景：地点 + 时间 + 一个看得见的物）
- 至少 1 处**反高潮**（不要每段都升华）
- 至少 1 处**口语词**（"咱"、"那啥"、"也是醉了"、"就离谱"）

## 禁用清单

| 禁用 | 因为 |
|------|------|
| 综上所述 / 总而言之 | 公文 |
| 在 X 的时代 / 在 X 的背景下 | 八股开场 |
| 不仅 X 而且 Y | 工整就死 |
| 让我们一起 X | 鸡汤 |
| 未来可期 / 拭目以待 | 万能空话 |
| 希望以上 ... 有所启发 | ChatGPT 出厂味 |
| em-dash 单段 > 1 | AI 标志 |
| 加粗滥用 | 加粗只用在 1-2 个金句上 |

## 鼓励的手法

- **画面开场**
  - ✅ "周五下午 4 点，李雷盯着电脑屏幕，手指悬在键盘上不动。"
  - ❌ "在当下竞争激烈的职场环境中，许多人面临选择 ..."

- **用对话**
  - ✅ "我同事说：'又改需求？' 我说：'第七次。'"

- **反高潮收尾**
  - ❌ "让我们一起拥抱变化，迎接更美好的未来！"
  - ✅ "事情就是这样。也没什么。"
  - ✅ "他后来怎么样了？没怎么样，还在那家公司。"

- **节奏短句**
  - ✅ "他没说话。走出去。门关上。整层楼安静了。"

## 节奏样本（写一段公众号像这样）

> 凌晨 1 点，我盯着 GitHub Actions 第 13 次失败的 log 看了半小时。
>
> 不是代码错。是 Mac M1 上 build 的镜像，跑到 GitHub 的 ubuntu-latest 上不认。
>
> 这事我前后查了三个晚上。Stack Overflow 翻烂。ChatGPT 问了 50 次。最后是同事一句话点醒：你 Dockerfile 写了 `--platform=linux/amd64` 没？
>
> 没写。
>
> 加上之后 18 秒过了 build。三个晚上。一行代码。
>
> 这种事我以前会写一篇深度长文复盘"如何避免跨平台 Docker 镜像问题"。现在不写了。这种坑就是要踩一次才记得住。
>
> 你也会的。

注意这段：

- 起手就是画面 + 时间
- 短句多，第 4 段就一个字
- 第一人称
- 有自嘲、有反高潮（"现在不写了"）
- 最后一句单独成段——节奏感
- 没有"综上所述"、没有"让我们一起"
- 唯一的总结句是"你也会的"——开放，不强行升华

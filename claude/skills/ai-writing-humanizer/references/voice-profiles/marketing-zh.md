# marketing-zh：高能带感型（营销 / 文案 / 产品推广 / 品牌稿）

目标读者：刷信息流的潜在用户，3 秒内决定要不要继续看。
参考调性：杜蕾斯文案、网易云音乐乐评、@阿球文案。

## 核心立场

- **3 秒钩子 + 画面 + 行动号召**
- **金句 > 论证**
- **画面 > 卖点列表**
- **反高潮和反差感是利器**

## 句法偏好

- **极短句开场**（3-8 字）作钩子
- 句子节奏起伏大，长短极不均衡
- 排比可以，但不要"X、Y、Z"这种三联词，要节奏不同的排比
- 段落短，1-3 句一段

## 必须出现

- 至少 1 个**金句**（短、有反差、可被截图）
- 至少 1 个**具体卖点**（数字 / 对比 / 时间）
- 至少 1 个**画面**（场景化使用情形）
- **行动号召清晰**（用户接下来该做什么）

## 禁用清单

| 禁用 | 因为 |
|------|------|
| 我们的产品 X 致力于 Y | 公文式自吹 |
| 引领 / 赋能 / 颠覆 / 革命 | 套话动词 |
| 强大 / 卓越 / 极致 / 领先 | 空形容词 |
| 不仅 X 而且 Y | 工整死 |
| 三联词（"快、准、稳"） | AI 标志 |
| 高效、稳定、可靠 | 直接搜公司官网的句子 |
| 让我们一起 | 鸡汤 |
| 感叹号狂轰滥炸 | 廉价 |

## 鼓励的手法

- **反差金句**
  - ✅ "她最骄傲的不是高跟鞋，是把它脱下来的样子。"
  - ✅ "深夜失眠的人，不是想太多，是没人能说。"
  - ✅ "好的工具不让你想起它存在。"

- **场景式开头**
  - ✅ "凌晨 2 点，你想喝水。冰箱开着。空了。"
  - ❌ "我们的产品致力于解决您日常生活中遇到的各类问题"

- **反高潮收尾**
  - ❌ "让我们一起拥抱美好生活，使用 X 吧！"
  - ✅ "你可以现在打开 App。也可以不。"

- **真实数字**
  - ✅ "我们公司 47 个人用了一年。平均每人每天省 23 分钟。"
  - ❌ "我们的产品广受好评"

## 节奏样本（写一段营销文案像这样）

> 周三下午 3 点。
>
> 老板转给你一个 PDF，说"看看这个我们做不做"，附 12 页排版。
>
> 你 Cmd+A 复制——这破 PDF 不能选文本。
>
> 截图。OCR。粘进 Notion。格式全乱。
>
> 这种时候你需要 X。
>
> 拖进去，3 秒。原文格式保留。表格还在。图片 alt 文字自动出来。
>
> 我们花了 14 个月做这件事。100% 因为我们自己每周遇到这破事 3 次。
>
> 现在它免费。试试，或者别试。

注意这段：

- 钩子是具体场景（不是"在数字化时代"）
- 短句 + 单字段落（"截图。OCR。粘进 Notion。"）
- 痛点真实（每个文案人都遇过）
- 卖点具体（"3 秒"、"格式保留"、"alt 文字自动"）
- 反高潮 CTA（"试试，或者别试"——不饥饿营销）
- 第一/第二人称（"你"、"我们"）
- 无套话开场、无套话收尾
